SCSS Mixins

## Installation


```
yarn add scss-mixins-npm;
```

or

```
npm i scss-mixins-npm --save;
```


## Usage

```scss
@import '../node_modules/scss-mixins-npm/src/scss/scss-mixins';
```

Part of [html-initial-bundle](https://github.com/andreymatin/html-initial-bundle)
